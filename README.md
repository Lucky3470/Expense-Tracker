# Expense Tracker (Spending Tracker)

A simple **Expense Tracker web application** built using **HTML, CSS, and JavaScript**.  
This project helps users **track their daily spending**, view total expenses, and delete expenses when needed.

---

## Features
- Add daily expenses (title & amount)
- View list of all expenses
- Automatically calculates **total money spent**
- Delete individual expenses
- Confirmation before deleting an expense
- Clean & minimal user interface
